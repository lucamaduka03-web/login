Budget Planner

Simple, offline-first budget planner built with vanilla HTML, CSS & JavaScript.

Open Live Demo Source / Files

Overview

Budget Planner is a lightweight tool to track income, expenses, and monthly balances. It works offline (localStorage) and focuses on clarity and accessibility.

Problem

Users needed a distraction-free way to track monthly spending without accounts or servers, and with easy export/reset options.

Goals

  • Minimal UI, mobile-first layout
  • Core flows: add/edit/delete transactions, monthly summary
  • Offline-first with export/import
  • Accessible controls and keyboard-first interactions

My role

Designer & frontend developer — researched, prototyped, implemented, and deployed the demo.

Tech & architecture

Vanilla HTML/CSS/JS, localStorage for persistence, small build (none), static deploy (GitHub Pages / Netlify).

Key interactions

  1. Add transaction modal (keyboard accessible)
  2. Inline edit + undo for last action
  3. Monthly roll-up summary with export CSV

Accessibility & performance

  • All interactive elements are keyboard focusable; forms have clear labels
  • Small asset sizes, images compressed, no external heavy libs
  • ARIA live regions for balance updates

Results

Simple, fast demo used for interviews and quick usability testing. Users reported the flows were intuitive and non-distracting.
