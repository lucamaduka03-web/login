Color Palette

Fast access to base colors with generated lighter & darker variants.

Open Live Demo Local Demo

Overview

The Color Palette tool generates accessible color scales from a base color and provides copyable CSS variables for quick use in designs and code.

Problem

Designers and developers waste time making consistent light/dark variants and ensuring contrast for accessibility.

Goals

  • Generate predictable lighter/darker shades
  • Provide easy copy of hex / rgba / CSS variable formats
  • Ensure adequate contrast and show WCAG hints

Role

Solo frontend developer — UX, implementation and deploy of the static demo.

Solution

A single-page app that accepts a base color, previews generated shades, shows contrast ratios, and exposes ready-to-paste variables. The UI is keyboard-first and mobile-friendly.

Key features

  • Auto-generated tints/shades with preview swatches
  • Contrast indicator with suggestions
  • Copy as CSS variables, hex, or rgba

Tech & Implementation

Vanilla JS + small utility for color transformations (HSL shifts). No build step required; static deployment on GitHub Pages / Netlify.

// generate shade (simplified)
function shade(h, s, l, delta){ 
  return `hsl(${h}, ${s}%, ${Math.max(0, Math.min(100, l + delta))}%)`;
}

Accessibility & Performance

  • Contrast warnings and WCAG pass/fail markers
  • Small assets, no external libraries, fast initial load

Results & Next steps

Useful tool for quick palettes; will add export presets and named-brand palettes next.
