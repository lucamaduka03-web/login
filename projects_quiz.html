Quiz-Game

A quick, responsive trivia game with score tracking and varied question sets.

Open Live Demo Local Demo

Overview

Responsive trivia quiz with timer, scoring, and category selection. Built for short-session play and quick sharing of results.

Problem

Users wanted a lightweight quiz to practice general knowledge without signups or heavy pages.

Goals

  • Fast game loop and clear feedback
  • Multiple categories and difficulty levels
  • Persistent top scores (local)

Role

Full-stack frontend: UX, game logic, and client-side persistence.

Solution

Single-page game with category selection, countdown timer, and result sharing. Questions are either embedded or fetched from a lightweight JSON endpoint.

Implementation highlight

// pick and shuffle
function pickQuestions(pool, n){
  return shuffle(pool).slice(0,n);
}

Accessibility & Performance

  • Controls accessible via keyboard; timer announced on change
  • Small payloads, images lazy-loaded, minimal DOM overhead

Results & Next steps

Fun, replayable demo; plan to add multiplayer pass-and-play and improved analytics.
