Password Generator

Create strong, memorable passwords with adjustable rules and copy shortcuts.

Open Live Demo Local Demo

Overview

The Password Generator helps users produce secure passwords and explains strength indicators while offering copy and entropy feedback.

Problem

Users struggle to create passwords that are both secure and memorable; default generator UIs often lack clarity about strength and rules.

Goals

  • Let users configure length and character sets
  • Provide clear strength and entropy feedback
  • Expose copy, pronounceable option, and reset

Role

Design + frontend implementation; focused on clarity and minimal dependencies.

Solution

Small SPA offering rule toggles, strength meter with ARIA live updates, and a pronounceable mode for memorability.

Implementation highlight

// entropy calc (simplified)
function entropy(length, poolSize){
  return (Math.log2(Math.pow(poolSize, length))).toFixed(1);
}

Accessibility & Performance

  • Strength meter announced via aria-live region
  • All controls keyboard reachable; copy uses Clipboard API
  • Lightweight, works offline

Results & Next steps

Meeting basic needs; plan to add password history and secure export/import features.
